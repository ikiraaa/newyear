# Сайт новогодних рецептов

Простой статический сайт с рецептами для новогоднего стола.

## Структура проекта

- `index.html` - Главная страница
- `recipes.html` - Страница с рецептами
- `tips.html` - Страница с советами
- `style.css` - Стили сайта
- `script.js` - JavaScript код

## Использование

1. Скопируйте все файлы в одну папку
2. Добавьте свои изображения в папку `images/`
3. Откройте `index.html` в браузере

## Особенности

- Адаптивный дизайн
- Плавная прокрутка
- Анимации элементов
- Новогодняя цветовая схема
